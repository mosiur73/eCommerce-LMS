# Plugins Used in This Project

The following plugins were used to build the functionality of this website:

| Plugin Name | Purpose |
| :--- | :--- |
| **WooCommerce** | Core eCommerce functionality and product management. |
| **Tutor LMS** | To create and manage courses, lessons, and quizzes. |
| **Elementor** | Page builder for custom UI/UX design. |
| **All-in-One WP Migration** | Used for exporting the complete site backup. |
| **Astra Sites** | For base theme and layout structure. |
| **WPForms Lite** | Used for contact or registration forms. |
| **Sbmart Core** | Theme-specific core functionalities. |
